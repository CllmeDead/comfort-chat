# Anonymous Comfort Chat

A slow, thoughtful space for anonymous human connection. No usernames, no profiles — just people offering each other comfort.

## Features

- **Send Comfort**: Share what's on your heart anonymously
- **Receive & Reply**: Read messages from others and respond with kindness
- **No Identity**: No usernames, emails, or profiles — just session tokens
- **Your Connections**: See conversations you've participated in

## Setup

1. Install Python dependencies:
```bash
pip install -r requirements.txt
```

2. Run the app:
```bash
python app.py
```

3. Open your browser to `http://localhost:5000`

## How It Works

1. You write a message and send it into the void
2. Someone else sees your message and replies with comfort
3. You can see the reply in your connections
4. You can also reply to others' messages
5. No one knows who anyone is — just that a human wrote back

## Tech Stack

- Python Flask (backend)
- SQLite (database)
- Vanilla HTML/CSS/JS (frontend)
- No external JS libraries — lightweight and fast

## Design Philosophy

- Warm, calming colors (cream, warm browns, soft greens)
- Serif fonts for reading comfort
- Generous whitespace
- Slow, intentional interactions
- No rush, no notifications, no pressure
